# Python Automated Network Scanner

## 📌 Overview
This project is a simple Python-based network scanner that:
- Pings devices on a local network
- Detects active hosts
- Scans for open ports (22, 80, 443)
- Exports results to a CSV file

## 🚀 Features
- Fast IP range scanning
- Port detection (SSH, HTTP, HTTPS)
- CSV output for easy analysis
- Cross-platform (Windows/Linux)

## 🛠️ Requirements
- Python 3.x

## ▶️ How to Run
```bash
python scanner.py
